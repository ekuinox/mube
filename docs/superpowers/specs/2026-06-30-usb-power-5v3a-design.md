# #23 USB 電源を 5V/3A 化（M-12001 採用）設計

- Issue: #23 C2 5V/1A USB 電源がサーボ失速時に不足する可能性（親 #12）
- 日付: 2026-06-30
- 種別: ドキュメント変更のみ

## 背景

5V/1A の USB AC アダプタ（M-08312）では、サーボ失速時に電流が不足する恐れがある。

| 負荷 | 電流 |
| --- | --- |
| Pico W（WiFi active） | 100-200mA |
| SG90 サーボ（失速） | 650-700mA |
| LED（330R, 3.3V） | 4mA |
| 合計 | 約 900mA（1A に迫る） |

サムターンが固い場合や突入電流を考えると 1A では足りない。`docs/parts-selection.md` に代替として記載済みの 5V/3A（M-12001）へ切り替える。

## 変更範囲

`docs/parts-selection.md` のみ。`circuit/netlist.py`・ファーム・筐体は変更しない。AC アダプタとケーブルは基板部品ではなく netlist 管理外（メイン表の手配線品目）であり、netlist の PARTS/BOM には現れないため。

ビルド・テストへの影響なし。検証は表の金額合計・品目数の算術整合の目視確認のみ。

## 決定事項

- 案 A 採用: M-12001（Micro-B オス直結 5V/3A ¥1,100）をメイン表へ昇格。Micro-USB 端子へ直結できるため、micro-USB ケーブル（C-09314 ¥150）は不要となり BOM から削除する。
- 旧 5V/1A 品（M-08312）はメイン表・代替候補とも完全削除する。issue の結論が「1A は不足」と明確なため、候補に残さない。
- 「USB AC アダプタは 5V/1A 以上」の注意書きは「5V/3A 推奨（最低でも 5V/2A 以上）」へ更新する。

3. 概算サマリ:
   - ① 理論按分合計 ¥3,274 → ¥3,454
